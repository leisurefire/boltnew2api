# 更新日志

## v1.1.0 (2025-11-28)

### ✨ 新功能
- 添加 API Key 认证支持（通过环境变量 `API_KEY` 配置）
- 添加 `claude-sonnet` 作为默认模型
- 增强的错误日志和调试信息
- 支持 Authorization Bearer Token 认证

### 🔧 改进
- 更详细的错误响应，包含错误详情
- 改进的请求日志，便于调试
- 更好的错误处理和状态码返回
- 优化了环境变量配置

### 📝 文档更新
- 更新了所有示例以包含 API Key 使用方法
- 添加了安全配置建议
- 更新了 OneAPI 集成说明
- 添加了测试脚本

### 🐛 修复
- 修复了 bolt.new API 请求错误的问题
- 改进了错误响应格式
- 修复了可能导致空响应的问题

## v1.0.0 (2025-11-27)

### 🎉 首次发布
- OpenAI 兼容的 API 端点
- 支持流式和非流式响应
- 从环境变量读取 cookies
- 支持个人和团队账号
- Zeabur 部署支持
- 健康检查端点
- 模型列表端点
